# Logan Simba — Portfolio Website

  Personal portfolio website built with HTML5, CSS3, and JavaScript.
  Features a responsive layout, dark/light theme toggle, smooth animations,
  and sections for projects, skills, and contact.

  Live site: [stupendous-cheesecake-aba26b.netlify.app](https://stupendous-c
  heesecake-aba26b.netlify.app/)

  ---

  ## Sections

  - **Hero** — intro and call to action
  - **About** — background, focus areas, and skills
  - **Projects** — featured work with tech stack and links
  - **Contact** — email, LinkedIn, and GitHub
